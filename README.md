## Clone the repo
  Git clone https://github.com/vishal098/Todo_app.git

## Installation

Use the Node package manager [npm](https://www.npmjs.com/get-npm) to install .

```bash
npm install
```
## env

## Start server
  Run this command on your terminal
